Form

关于表单结构的说明

form元素必须添加ocp-form-horizontal样式，form内部结构为：

最外层fieldset元素，div.control-group为一行字段，label.control-label为字段标题，div.control为字段区域，

	  <form class="ocp-form-horizontal">
		<fieldset>
		  <div class="control-group">
			<label class="control-label" for="input01">文本输入框</label>
			<div class="controls">
			  <input type="text" class="input-xlarge" id="input01">
			</div>
		  </div>
		</fieldset>
	  </form>

OCP 表单控件

( 即将上线功能： 定向优惠、抽奖、 投票、 掌柜说话题讨论区 )

可申请宝贝数量：件

包含哪些控件

左侧即是OCP-DPL支持的所有默认表单控件:

  • 文本输入框 (文本，密码，邮件等等)
  • 复选框
  • 单选框
  • 下拉框
  • 多选框
  • 上传框
  • 文本域

OCP Form中对input约定了不同长度的专属样式

.input-smini.input-mini.input-small.input-medium等。

单选框和复选框

罗列单选框和复选框时，要将它嵌套到到其他标记内。 只须用 <label class="checkbox">、<input type="checkbox"> 包含即可。 也支持行级单选框和复选框。只要对 .checkbox 或 .radio 应用 .inline 即可。

标注提示信息

再另一行标注提示信息时。 只须用 <p class="help-block"> 或 <p class="help-inline"> 包含即可。

表单操作区域

提交按钮等所在的操作区域， 只须用 <div class="form-actions"> 包含即可。

bubble

OCP中的常用信息气泡样式.

关于箭头的说明

箭头的实现使用的是两个宽高均为0的i元素，叠加，并设置其border来实现。箭头的大小和border宽度均可以再源码中设置

箭头的具体位置可以通过手动设置.bubble-arrow的left、right、bottom、top值来实现

使用箭头必须设置箭头的位置，四个方向的class分别为：bubble-arrow-top、bubble-arrow-left、bubble-arrow-right、bubble-arrow-bottom

一般的HTML结构如下：也可以根据需要添加多个箭头，只需要多设置几个bubble-arrow便可

                    <div class="bubble">
                        <div class="bubble-arrow bubble-arrow-top"><i></i><i></i></div>
                        这是一个小气泡
                    </div>
